## Por qué este sistema

El Tehilim —los Salmos de David— se ha transmitido en hebreo durante más de dos mil años. Esta web nace para que el hispanohablante —judío de nacimiento, converso o baal teshuvá— pueda rezarlos, estudiarlos y comprenderlos sin perder la conexión con el texto original.

Para eso hacen falta dos herramientas: una **transliteración fiel a la pronunciación sefaradí**, y una **traducción honesta con el hebreo y con la tradición judía**.

---

## Sistema de transliteración — Nusaj Sefardí

La pronunciación usada es la sefaradí tradicional, que es también la base del hebreo moderno. No es la pronunciación ashkenazí (que convierte la תּ en "s" o la חֲ en "kh"). Aquí la ת siempre es "t", la ח siempre es "j" (como la jota española), y no se distingue entre kamatz y pataj.

### Vocales

| Vocal | Nombre | Se lee |
|---|---|---|
| ַ | Pataj | a |
| ָ | Kamatz | a |
| ֶ | Segol | e |
| ֵ | Tsere | e |
| ִ | Jirik | i |
| ֹ | Jolam | o |
| וּ | Shuruk | u |
| ֻ | Kubuts | u |
| ְ | Shvá na | e (breve) |
| ְ | Shvá naj | silencio |

El **shvá na** suena como una "e" muy breve al inicio de sílaba. El **shvá naj** es silencio total.

### Consonantes principales

| Letra | Se lee | Nota |
|---|---|---|
| בּ | b | con daguesh |
| ב | v | sin daguesh |
| ח | j | jota española, gutural |
| כּ | k | con daguesh |
| כ / ך | j | sin daguesh |
| פּ | p | con daguesh |
| פ / ף | f | sin daguesh |
| צ / ץ | ts | siempre "ts" — nunca "z" sola |
| שׁ | sh | |
| שׂ | s | |
| ת | t | **siempre "t"** — nunca "s" |
| ר | r | "r" simple española |

### Reglas prácticas

- El **acento** se marca con tilde sobre la sílaba tónica: Torá, berajá, Elohím.
- El **maqaf** (־) se mantiene como guión: ashré-haish.
- El **artículo** (הַ) se une a la palabra: haish, hayóm.
- Los **prefijos** no se separan del cuerpo: ve-lo, berajá, uleolám.

## Sistema de traducción

### Los Nombres de Dios

Este es el punto más delicado. En la tradición judía los nombres de Dios no son intercambiables — cada uno refleja un atributo distinto.

| Hebreo | Traducción | Por qué |
|---|---|---|
| יהוה | **El Eterno** | Refleja la eternidad del Ser sin pronunciar el Nombre sagrado. Traducción clásica de la tradición judía (Hirsch, Kahn). Nunca "Señor" ni "LORD" — esas son traducciones cristiana y protestante. |
| אֱלֹהִים | **Dios** | Nombre de poder y juicio universal |
| אֵל עֶלְיוֹן | **El Altísimo** | |
| שַׁדַּי | **Shadai** | Se translitéra — es intraducible |

### Vocabulario con traducción fija

Algunos términos hebreos tienen una carga semántica que no existe en español. Se mantienen en hebreo transliterado o se traducen siempre de la misma manera para crear consistencia:

| Hebreo | Traducción | Por qué no otra |
|---|---|---|
| תּוֹרָה | **Torá** | Nunca "ley" — esa traducción, heredada del griego *nomos*, tiene connotaciones peyorativas en el contexto cristiano que son ajenas al judaísmo |
| חֶסֶד | **bondad leal** | Es el amor comprometido, leal a la alianza. "Misericordia" es demasiado cristiano. "Amor" es demasiado vago |
| נֶפֶשׁ | **alma** | No "vida" — eso es חַיִּים |
| בְּרִית | **alianza** | No "pacto" ni "testamento" |
| צַדִּיק | **justo** | |
| רָשָׁע | **malvado** | No "impío" — que suena arcaico y cristiano |
| סֶלָה | **Selá** | Se translitéra siempre — es una pausa litúrgica cuyo significado exacto no está definido |
| הַלְלוּיָהּ | **Haleluya** | Se translitéra siempre |

### Nombres propios

Se usan las formas hebreas transliteradas para que el lector aprenda los nombres correctos:

**Moshé** (no Moisés) · **Shelomó** (no Salomón) · **Avraham** · **Tsión** · **Yerushaláyim**

### Principios generales

**Fidelidad estructural.** El Tehilim es poesía con paralelismo bíblico: dos hemistiquios que se responden. La traducción respeta esa estructura — cada unidad de sentido es una línea.

**Tiempo presente.** Los salmos se rezan, no se narran. Salvo los salmos históricos (78, 105, 106), se traduce en presente.

**Mayúsculas para lo Divino.** Todo pronombre que se refiere a Dios va en mayúscula: Tu, Tuyo, Él, Su, Te, Ti.

**Tono elevado pero claro.** Los salmos son texto sagrado — se acepta un tono levemente formal ("dad", "ved", "cantad"). Pero nunca a costa de la comprensión.

**Literalidad con dignidad.** La prueba: ¿puede un lector seguir el hebreo verso a verso con esta traducción? Si sí, está bien. Si la traducción se entiende sola sin el hebreo, se ha alejado demasiado.
